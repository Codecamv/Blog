Embedding EPiServer VPP images in HTML using base64 encoding
https://blog.mathiaskunto.com/2011/08/21/embedding-episerver-vpp-images-in-html-using-base64-encoding/

Version 1.1.0.0

******************************************************************************************

Usage:

* Include the Base64EmbeddedImage.csproj in your Visual Studio solution.
* Reference it from your web project.
* Make sure all EPiServer references are OK in the Base64EmbeddedImage project.

Note:
Namespaces and so on are not the same as in the blog post examples after the move to a separate project in v1.1.0.0. Example code is not included in droppable binary version; if you need this, you should get the zipped source code or go to GitHub: https://github.com/matkun/Blog/tree/master/Base64EmbeddedImage
